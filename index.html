What i do

I have more than 10 years of experience in the field of information technology. Below is a quick overview of my main technical skill sets and technologies I use. Want to find out more about my experience? Check out my online resume and project portfolio.
HTML & CSS

As a full stack developer, I have high skills in HTML and CSS. And my favorite CSS framework is Bulma.

JavaScript & Vue.js

I use JavaScript and Vue.js framework to develop SPA web applications, and Nuxt.js for cases where SEO is a priority.

PHP & Laravel

The back-end of most of the projects I developed was with PHP and the popular Laravel framework. I am good at them.

Python & FastAPI

I use Python and Django or FastAPI frameworks in some of my new projects. I personally prefer Python to PHP.

SQL & NoSQL

I have a high command of MySQL and MariaDB. But if the project requires, I have the ability to work with MongoDB.

Network

As a senior developer, I am familiar with network concepts and various protocols and services.

Linux and Services

I have been using Linux for many years and I have the ability to setup Web, FTP, Email, DNS, etc services.

Git & Github

Of course, every developer should be able to work with Git, GitHub or GitLab. Sometimes I also have a local git server.
